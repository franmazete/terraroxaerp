# Logos das empresas

Salve os 2 PNGs aqui com estes nomes exatos:

| Arquivo | Empresa | Onde aparece |
|---|---|---|
| `terra-roxa.png` | Terra Roxa Comércio de Cereais (oliveiras + texto vinho) | Header padrão, empresa origem **5** |
| `tr-trading.png` | TR Trading Exportação LTDA (TR + globo + seta amarela) | Header alternativo, empresa origem **6** |

## Especificação técnica recomendada

- **Formato**: PNG com fundo transparente
- **Altura**: 64px ou maior (vamos reescalar)
- **Largura**: proporcional (não cortar)

Enquanto o arquivo não existir, o componente `<Logo>` cai num fallback de texto estilizado.
